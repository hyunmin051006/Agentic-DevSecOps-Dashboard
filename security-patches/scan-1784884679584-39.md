# Missing Security Header: content-security-policy

- **Severity**: Medium
- **OWASP Category**: A05:2021 - Security Misconfiguration
- **Source**: https://pro.mincoding.co.kr/

## Description
Content-Security-Policy 헤더가 없어 XSS 등 콘텐츠 주입 공격의 피해 범위를 제한하지 못합니다.

## Before
```
// 응답 헤더 없음
```

## Suggested Fix (After)
```
content-security-policy: <appropriate-value>
```

_이 문서는 보안 대시보드가 자동 생성한 패치 노트입니다. 저장소에서 대응하는 소스 파일을 찾지 못했거나(예: 웹사이트 스캔으로 발견된 항목) 코드 내용이 일치하지 않아 별도 문서로 커밋되었습니다._
