# Missing Security Header: x-frame-options

- **Severity**: Medium
- **OWASP Category**: A05:2021 - Security Misconfiguration
- **Source**: https://pro.mincoding.co.kr/

## Description
X-Frame-Options 헤더가 없어 클릭재킹(Clickjacking) 공격에 노출될 수 있습니다.

## Before
```
// 응답 헤더 없음
```

## Suggested Fix (After)
```
x-frame-options: <appropriate-value>
```

_이 문서는 보안 대시보드가 자동 생성한 패치 노트입니다. 저장소에서 대응하는 소스 파일을 찾지 못했거나(예: 웹사이트 스캔으로 발견된 항목) 코드 내용이 일치하지 않아 별도 문서로 커밋되었습니다._
